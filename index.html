처리할 활성영역 선택
필터는 아래 모든 그래프·표·통계에 동시에 적용됩니다

위도별 회전 각속도 Ω(φ)

각 점은 활성영역 하나 — Stonyhurst(hgs_x) 경도의 시간 변화 기울기를 보정 없이 그대로 쓴 삭망(지구 관측) 회전율입니다. 점 크기는 추적 신뢰도(기간/RMS).

활성영역 경도·위도 추적

Carrington 경도의 시간 변화 기울기가 곧 차등회전 신호입니다 (기울기 + 14.1844 = Ω).

아래 경도 그래프의 점을 클릭하면 선형 피팅에서 제외/포함할 수 있습니다 — 어떤 점을 쓰느냐에 따라 Ω가 달라집니다.

태양면 위 활성영역 위치

지구에서 본 태양면(정사영). 날짜를 움직이면 흑점들이 동→서(왼→오른쪽)로 회전해 가는 것이 보입니다.

데이터 표

모든 값은 그래프와 동일한 계산 결과입니다. 열 머리글을 누르면 정렬됩니다.

일치 화면 AR(Helioviewer 11개)이 2026-05-29에 가졌던 hgc(Carrington 경도) 값과 ±° 안에 드는 모든 행 — AR·날짜 무관하게 hgc 컬럼 값만 비교합니다. 행에 마우스를 올리면 어떤 화면 AR과 일치하는지 보입니다.

방법론 — 어떻게 계산했나

1. 데이터. NASA HEK(Heliophysics Event Knowledgebase)에서 활성영역(AR) 검출을 수집합니다 — Helioviewer 화면 마커의 원천 데이터와 동일합니다. 검출 소스는 HMI SHARP(SDO/HMI 위성 자동 파이프라인)와 NOAA SWPC Observer(사람이 분석한 보고) 둘이며, 상단 데이터 소스 선택으로 둘 다·HMI만·NOAA만 중 고를 수 있습니다. 림 투영오차를 피하려고 |Stonyhurst 경도| ≤ 60° 검출만 쓰고, 선택한 소스에 대해 NOAA 번호·날짜별 중앙값으로 일별 위치를 만듭니다 (소스를 바꾸면 중앙값이 다시 계산되어 각속도도 조금씩 달라집니다).

2. 회전 각속도. Carrington 좌표계는 태양과 함께 항성 기준 ΩC = 14.1844°/일로 회전합니다. 위도 φ의 흑점이 국소 회전율 Ω(φ)로 돌면 Carrington 경도가 그 차이만큼 표류합니다:

dLC/dt = Ω(φ) − ΩC  ⇒  Ω(φ) = 14.1844 + (LC(t) 선형피팅 기울기)

좌표계를 hgs(Stonyhurst)로 바꾸면 지구 공전 보정 없이 hgs_x(중앙자오선 기준 경도)의 시간 변화 기울기를 그대로 Ω로 사용합니다 — 이건 항성 회전율이 아니라 지구에서 본 삭망(synodic) 회전율입니다.

3. 차등회전 프로파일. AR별 (φ, Ω)에 대해 추적 신뢰도 w = 기간/(RMS+0.1) 가중 최소제곱으로 고전적 프로파일을 피팅합니다:

Ω(φ) = A + B·sin²φ  (+ C·sin⁴φ)

4. 검증. hgc 기반 결과는 독립적인 Stonyhurst 경도법(삭망 회전율에 지구 공전분 +0.986°/일을 더해 항성 변환한 값)과 0.1–0.4°/일 이내로 일치했으며, 적도 회전율은 문헌값(Snodgrass & Ulrich 1990: Ω = 14.71 − 2.39sin²φ − 1.78sin⁴φ, 항성 기준)과 부합합니다. sin²φ 계수가 문헌보다 가파른 것은 흑점 고유운동 산란과 위도 범위(±33°) 제한 때문입니다.

품질 필터. 피팅 RMS ≤ 1.0°(끄고 켤 수 있음), 관측일 ≥ 5(슬라이더), 추적기간 ≥ 5일. 이 3가지 외의 조건(예: 회전율 범위 제한)은 두지 않습니다 — 트랙 생성 전 사전 컷(일별검출 ≥ 3일, 추적 ≥ 2일)만 별도로 적용됩니다.

AR 번호 · 자기장 분류 기호 — 읽는 법

활성영역(AR) 번호. NOAA/SWPC는 흑점군이 반음영-본음영 구조를 갖춘 정도로 조직화됐다고 판단될 때 새 번호를 순차적으로 부여합니다 (아주 작은 pore 단계에서는 번호를 붙이지 않음). 번호는 계속 증가만 하며 재사용되지 않습니다 — 같은 물리적 흑점군이라도 자전으로 태양 뒷면을 돌고 다시 앞면(동쪽 림)으로 나타나면 새 번호를 받습니다. 즉 AR 번호는 흑점의 물리적 정체성이 아니라 지구에서 연속 관측된 구간을 가리키는 식별자입니다.

자기장 분류(Hale/Mount Wilson). "NOAA 14455 β"처럼 번호 옆에 붙는 기호는 흑점군의 자기극성 복잡도이며 플레어 발생 가능성과 상관됩니다.

α 단일 지배 극성(단순)  ·  β 양극성이 뚜렷이 분리  ·  γ 양·음극이 뒤섞여 경계 불명확
βγ 부분적으로 감마 특성을 가진 베타  ·  βδ, βγδ — δ(반대 극성 본음영이 같은 반음영 안에 10° 이내로 인접) 포함, 대형 플레어 위험 최고

복잡도는 대략 α < β < βγ < γ < βδ < βγδ 순으로 커집니다.
